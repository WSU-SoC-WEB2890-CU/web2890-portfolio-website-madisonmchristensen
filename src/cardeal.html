CarDeal

Project Overview

In this project I created a site for a business that sells cars. I used BootStrap to create this one-page website.

Tools Used

  • Visual Studio Code
  • HTML, CSS, SASS
  • Bootstrap
  • Canva

Details and Findings

I enjoyed creating images and playing with fonts and design elements such as gradients. I liked making the form a blur background to see the image underneath.

Key Takeaway

I learned that while I enjoyed making this website, the feedback I received made me realize a lot of my colors and text clashed. I also struggled to get some of the elements to not be full screen.
